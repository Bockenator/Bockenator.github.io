PRE-PARALLAX


Visualisation of Repair Process of Oligodendrocyte Precursor Cells


There are many neurological processes that aren't fully understood, even those that are can be difficult to conceptualise. As a result visual aids to give a broad and general picture of what we think might be happening in a certain processes. One such process is the behaviour exhibited by Oligodendrocyte Precursor Cells (OPCs) in response to damage to an area of the brain. More specifically, these cells respond to the chemical signals emitted by the damaged area. The primary goal of the OPC is to repair the myelin sheath around axons present in the brain by duplicating and differentiating into full Oligodendrocyte. The myelin sheath is what allows saltatory conduction along an axon (this is what allows signals to quickly move along the axis). A disruption to this conduction (for instance by some demyelinating process) causes a many problems including physical movement issues and psychological and emotional difficulties. One such demyelinating disease is Multiple Sclerosis which is presumed to affect over 100000 people in the UK alone (NHS). As a result understanding why the disease occurs and the natural response of biological cells to these diseases is of great interest.


This project aims to give a basic insight into how OPCs behave when a lesion is present in the brain. The data for the visualisation is obtained from a stochastic model built using MELA. The visualisation on this page takes a particular instance of this language (found here)which models static and moving OPCS, mature OPCs(Oligodendrocytes) and lesion areas in the brain. The MELA model can output several meta data files, including a file which describes every action taken at every time step. This meta file is parsed by a python script and interpolated so that more meaningful visual data can be extracted from the meta files. The newly created file is what is played by the javascript contained on this page.
